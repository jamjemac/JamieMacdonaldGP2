# JamieMacdonaldGP2

Model viewer program created in C++ \& OpenGL



## How to operate

* Up/Down arrow keys - Used to scale the current object being shown to be either bigger or smaller.
* WASD Keys - Used to rotate the current object being show in the direction of the key being pressed.
* Left/Right arrow keys - Used to change the model currently being displayed.



## How to enable/disable shaders

* Press the 1 key to disable the ADS/Rim lighting shader and replace it with a basic texture shader.
* Press the 2 key to reenable the ADS/Rim lighting shader.



## Models / Audio

* All models - Kenney's Game Assets
* Sound effect - Kenney's Game Assets
* Background Music - Nintendo (Super Smash Bros. Melee)
